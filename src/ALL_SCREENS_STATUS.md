## HOW TO CONVERT REMAINING SCREENS

### Step 1: Update Imports
```tsx
// Replace this:
import { MobileContainer } from '../components/MobileContainer';
import { Header } from '../components/Header';

// With this:
import { DesktopLayout } from '../components/DesktopLayout';
```

### Step 2: Update JSX
```tsx
// Replace this:
<MobileContainer>
  <Header title="Screen Title" />
  <div className="p-4">
    {/* content */}
  </div>
</MobileContainer>

// With this:
<DesktopLayout title="Screen Title">
  <div className="max-w-6xl mx-auto">
    <div className="bg-white rounded-xl shadow-sm border border-gray-200 p-8">
      {/* content */}
    </div>
  </div>
</DesktopLayout>
```

### Step 3: Enhance Styling
- Increase padding: `p-4` → `p-8`
- Add max-width: Wrap in `max-w-6xl mx-auto`
- Enhance cards: Add `rounded-xl shadow-sm border`
- Increase text size: `text-base` → `text-lg`
- Improve headings: `text-xl` → `text-3xl`
